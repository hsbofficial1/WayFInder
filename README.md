# WayFinder - Indoor Navigation

WayFinder is a manual indoor navigation app designed to provide step-by-step walking directions inside buildings without relying on GPS or complex sensors.

## Features

- **Step-by-Step Navigation**: Clear instructions to guide you from point A to point B.
- **Offline Capable**: Works without constant internet connection once loaded (PWA).
- **Mobile Friendly**: Designed as a Progressive Web App (PWA) for a native-like experience on mobile devices.

## Technology Stack

- **Frontend**: React, TypeScript, Vite
- **UI Framework**: shadcn-ui, Tailwind CSS
- **Routing**: React Router
- **State Management**: React Query
- **PWA**: vite-plugin-pwa

## Getting Started

To run this project locally:

1.  **Clone the repository**:
    ```sh
    git clone <YOUR_GIT_URL>
    cd WayFinder
    ```

2.  **Install dependencies**:
    ```sh
    npm install
    ```

3.  **Start the development server**:
    ```sh
    npm run dev
    ```

4.  **Build for production**:
    ```sh
    npm run build
    ```
